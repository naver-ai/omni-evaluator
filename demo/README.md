# OmniEvaluator Demo

OmniEvaluator is a composable evaluation system for reproducible omni-modal foundation model evaluation. This demo showcases the core workflows of OmniEvaluator across three scenarios.

</div>

## [Demo Videos](./full_demo.mp4))

### 1. Local Evaluation

CLI-based evaluation on a locally configured environment. Users specify the inference engine, evaluation framework, and benchmark through a single command, and OmniEvaluator produces a provenance-rich evaluation artifact upon completion.

▶️ [Local Evaluation Demo](./local_evaluation.mp4)

### 2. Remote Evaluation

Server-based evaluation via REST API. An administrator launches a persistent evaluation server, and team members submit evaluation requests without local environment setup—ensuring identical inference environments across all users.

▶️ [Remote Evaluation Demo](./remote_evaluation.mp4)

### 3. Dashboard

An integrated dashboard that synchronizes with evaluation artifacts, providing cross-modal benchmark result visualization, Omni Score computation for model selection, and side-by-side sample comparison across experiments.

▶️ [Dashboard Demo](./dashboard.mp4)
