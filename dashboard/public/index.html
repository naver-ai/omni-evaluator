Overview

Abstract

Foundation models are increasingly omni-modal, yet their evaluation remains fragmented across modality-specific toolchains, prompt conventions, and metric implementations, making comprehensive assessment costly and cross-paper comparison unreliable. We present OmniEvaluator, a composable evaluation system that makes existing evaluators interoperable through a shared intermediate schema without reimplementation, while every run produces a provenance-rich artifact that fully captures the configuration needed for reproduction. The system currently integrates four inference backends and five evaluation frameworks covering over 2,000 benchmarks spanning text, image, video, and audio. On top of this architecture, OmniEvaluator provides a unified omni-modal leaderboard with weighted scoring for model selection, an LLM-as-a-judge mechanism that normalizes heterogeneous metrics onto a comparable scale, and a federated evaluation pipeline for efficient GPU utilization.
