# Supported Tasks

Full task catalog for OmniEvaluator, grouped by evaluation engine. Back to the [README](../README.md).

Counts are **deduplicated by base benchmark**. Evaluation-protocol variants (`_zeroshot`, `_cot`, `_n_shot`, `_generative`, `_fewshot`, `_direct`, …) and video-frame variants (`_8frames`, `_64frames`, `_128frames`) collapse to a single entry. Subset markers that name a **different** benchmark (`_pro`, `_redux`, `_hard`, `_diamond`, `_extended`, `_main`, `_mini`, `_v2`, …) are preserved as independent entries. Use the CLI for the raw (undeduped) live list:

```bash
python run.py list --tasks --evaluation_engine="<engine_name>"
```
